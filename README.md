# 🌱 佛系种地 (Zen Farm)

一款极简佛系种植微信小游戏。没有压力，没有任务，只有你和你的植物。

## 游戏理念

- **真实自然规律** — 植物按真实时间生长、成熟、衰老、死亡
- **养成即扩展** — 一棵植物养成后，获得新花盆，可以选择种新的
- **照顾有代价** — 所有植物都需要浇水照顾，不照顾就按自然规律走（枯萎、死亡）
- **自由选择** — 可以只守一棵，也可以慢慢建花园；可以领新盆，也可以不领
- **所有种子免费** — 没有金币，没有内购，纯粹的种植体验

## 核心玩法

### 机制
1. **起步** — 1 个花盆，选一种植物播种
2. **照顾** — 定期浇水，植物按真实时间生长
3. **养成** — 植物成熟后，可以领取一个新花盆（可选，不强制）
4. **扩展** — 新花盆可以种新植物，但每棵都需要照顾
5. **自然规律** — 不浇水→枯萎→死亡，花盆空出来
6. **挖掉重来** — 随时可以挖掉任何一棵，花盆空出来种新的

### 生命周期
```
播种 → 发芽 → 生长 → 成熟 → 持续产出 → 衰老 → 死亡
```
- 每个阶段都有视觉变化
- 不浇水会加速进入衰老/死亡
- 成熟期间可以收获果实

### 玩家风格
- **佛系** — 只养一棵，专心陪伴
- **收集** — 不断领新盆，建大花园（但照顾压力大）
- **中间** — 养几棵就好，够看就行

### 植物列表（Phase 1）

| 植物 | 生长周期 | 寿命 | 产出间隔 |
|------|---------|------|---------|
| 🥬 小白菜 | 7 天 | 30 天 | 3 天 |
| 🥕 胡萝卜 | 30 天 | 90 天 | 7 天 |
| 🍅 番茄 | 45 天 | 120 天 | 5 天 |
| 🌽 玉米 | 60 天 | 150 天 | 10 天 |
| 🍉 西瓜 | 90 天 | 200 天 | 15 天 |
| 🌻 向日葵 | 120 天 | 300 天 | 20 天 |
| 🌸 樱花树 | 365 天 | 数年 | 30 天 |

> 生长周期 = 种子到第一次成熟；寿命 = 自然死亡时间（照顾好可延长）；产出间隔 = 成熟后每次收获的间隔

### 成长阶段（视觉）
1. **种子** — 土里一个小坑
2. **发芽** — 冒出小绿芽（~20%）
3. **幼苗** — 长出叶子（~40%）
4. **成长** — 明显长大（~70%）
5. **成熟** — 开花/结果，可收获
6. **衰老** — 叶片发黄，产出减少
7. **枯萎/死亡** — 干枯，花盆空出

### 浇水机制
- 每天需要浇水（每棵植物独立）
- 浇水：正常生长
- 不浇水 1 天：没事
- 不浇水 3 天：开始枯萎
- 不浇水 7 天：死亡
- 下雨天自动浇水（随机事件）

## 技术方案

### 引擎
- **Cocos Creator 3.x** — 微信小游戏官方支持，动画系统成熟

### 前端架构
```
zen-farm/
├── assets/
│   ├── scenes/       # 场景文件
│   ├── scripts/      # TypeScript 脚本
│   │   ├── GameManager.ts    # 游戏主逻辑
│   │   ├── PlantManager.ts   # 植物生长管理（多花盆）
│   │   ├── PlantData.ts      # 植物数据定义
│   │   ├── TimeManager.ts    # 时间计算（真实时间）
│   │   ├── WaterManager.ts   # 浇水/枯萎逻辑
│   │   └── UIManager.ts      # UI 控制
│   ├── textures/     # 植物和场景贴图
│   └── animations/   # 生长动画
├── build-templates/
│   └── wechatgame/   # 微信小游戏模板
└── settings/
```

### 数据存储
- **本地存储**（wx.setStorageSync）：
  - 花盆列表（每盆：植物类型、播种时间、上次浇水时间、生长状态）
  - 累计收获记录
  - 可用空花盆数量
- **Phase 1 纯本地**，不需要后端
- 后续可加云存储（换设备同步）

### 时间计算
```typescript
// 打开游戏时计算每棵植物状态
const now = Date.now()
const daysSincePlant = (now - plantedAt) / (1000 * 60 * 60 * 24)
const daysSinceWater = (now - lastWateredAt) / (1000 * 60 * 60 * 24)
const progress = Math.min(daysSincePlant / growthCycleDays, 1.0)
const isWilting = daysSinceWater > 3
const isDead = daysSinceWater > 7
```
