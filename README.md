# 🌱 佛系种地 (Zen Farm)

一款极简佛系种植微信小游戏。没有压力，没有任务，只有你和一棵植物。

## 游戏理念

- **一块地，一棵植物** — 一次只能种一种，想换就得挖掉重来
- **真实生长周期** — 萝卜 30 天，白菜 60 天，西瓜 90 天……像养真植物一样等待
- **收获后继续生长** — 不用重新播种，它会一直长，一直收
- **零压力** — 不浇水也不会死，不排名，不内卷

## 核心玩法

### 种植循环
1. **选种** — 从已解锁的种子中选一种
2. **播种** — 种到地里，开始等
3. **浇水**（可选）— 偶尔浇水，加速一点点
4. **等待** — 每天打开看看，像去阳台看一眼
5. **收获** — 成熟了就收，获得金币，植物继续生长
6. **换种？** — 挖掉当前植物，放弃所有等待，从种子重来

### 植物列表（Phase 1）

| 植物 | 生长周期 | 收获间隔 | 卖价 | 种子价 |
|------|---------|---------|------|--------|
| 植物 | 生长周期 | 收获间隔 |
|------|---------|---------|
| 🥬 小白菜 | 7 天 | 3 天 |
| 🥕 胡萝卜 | 30 天 | 7 天 |
| 🍅 番茄 | 45 天 | 5 天 |
| 🌽 玉米 | 60 天 | 10 天 |
| 🍉 西瓜 | 90 天 | 15 天 |
| 🌻 向日葵 | 120 天 | 20 天 |
| 🌸 樱花树 | 365 天 | 30 天 |

> 所有种子免费，想种什么种什么。核心是"选择"和"陪伴"，不是收集和解锁。
> 生长周期 = 从种子到第一次成熟的时间
> 收获间隔 = 收获后到下一次成熟的时间

### 成长阶段

每种植物 5 个视觉阶段：
1. **种子** — 刚播种，土里一个小坑
2. **发芽** — 冒出小绿芽（约 20% 生长周期）
3. **幼苗** — 长出叶子（约 40%）
4. **成长** — 明显长大（约 70%）
5. **成熟** — 开花/结果，可以收获

### 佛系特色
- 不浇水：生长速度 ×1（正常）
- 浇水：生长速度 ×1.1（快一丁点，聊胜于无）
- 每天最多浇 3 次水
- 下雨天自动浇水（根据真实天气？或随机）
- 植物不会枯萎、不会死

- **Cocos Creator 3.x** — 微信小游戏官方支持，动画系统成熟

### 数据存储
- **本地存储**（wx.setStorageSync）：
  - 当前种植的植物类型
  - 播种时间戳
  - 上次浇水时间
  - 累计收获次数
  - 累计收获列表（品种+次数）
- **Phase 1 纯本地**，不需要后端
- 后续可加云存储（换设备同步）

### 时间计算
```typescript
// 核心逻辑：打开游戏时计算植物状态
const now = Date.now()
const plantedAt = storage.get('plantedAt')
const growthDays = plant.growthCycleDays
const elapsed = (now - plantedAt) / (1000 * 60 * 60 * 24) // 天数
const progress = Math.min(elapsed / growthDays, 1.0) // 0~1
const stage = getStageFromProgress(progress) // 种子/发芽/幼苗/成长/成熟
```
