# 🌱 佛系种地 (Zen Farm)

一款真实模拟种植的微信小游戏。联动当地真实天气，按自然规律生长。

## 游戏理念

- **真实自然规律** — 温度、降水、日照、土壤湿度全部影响植物状态
- **当地天气联动** — 读取用户所在地的实时天气数据
- **真实生长过程** — 植物从种子到死亡，每个阶段都模拟真实反应
- **难度即真实** — 幸运草好养，草莓难养，因为现实就是这样
- **所有种子免费** — 没有金币，没有内购

## 核心机制

### 环境系统（核心）
游戏读取用户当地的真实天气数据，计算以下环境参数：

| 参数 | 来源 | 影响 |
|------|------|------|
| 气温 | 天气 API 实时数据 | 生长速度、热害/冻害 |
| 降水 | 天气 API（是否下雨/雪） | 土壤湿度自动增加 |
| 日照 | 根据天气状况推算（晴>多云>阴>雨） | 光合作用、向日葵等喜阳植物 |
| 土壤湿度 | 动态计算（降水+浇水-蒸发） | 0~100%，过低=旱，过高=涝 |

**土壤湿度计算：**
```
湿度变化 = +浇水量 + 降水量 - 蒸发量
蒸发量受气温和日照影响（高温+晴天 → 蒸发快）
```

### 植物健康系统
每棵植物有**健康状态**，按真实过程变化：

```
健康 → 轻微受损 → 明显受损 → 严重衰弱 → 死亡
```

- 状态变化有**真实的时间过程**，不会瞬间跳变
- 不同植物对不同胁迫的耐受时间不同
- 恢复也需要时间——条件好转后不会立刻变健康
- 每个状态有对应的视觉表现（叶色、姿态、果实等）

### 植物列表（Phase 1：4 种）

#### 🍀 幸运草（白三叶）— 难度 ★
- **适宜温度**：10~25°C
- **耐热**：30°C 以上生长变慢，35°C 以上叶片焦边
- **耐寒**：耐寒到 -5°C，低于此叶片冻伤但根部可活
- **需水**：低，耐旱；土壤湿度 20~70% 均可
- **日照**：不挑，半阴到全日照都行
- **生长周期**：14~21 天成熟
- **寿命**：多年生，照顾好可以活很久
- **真实反应**：
  - 干旱：叶片卷曲→发黄→枯萎（耐旱，过程慢）
  - 高温：叶缘焦枯→全叶萎蔫
  - 积涝：根部腐烂（但需要长时间积水）

#### 🌻 向日葵 — 难度 ★★
- **适宜温度**：18~30°C
- **耐热**：喜温，35°C 以上花盘发育不良
- **耐寒**：不耐寒，10°C 以下停止生长，0°C 冻死
- **需水**：中等，根系深；土壤湿度 30~60%
- **日照**：需要充足阳光，连续阴天影响大
- **生长周期**：60~90 天开花
- **寿命**：一年生，开花结籽后自然衰老死亡
- **真实反应**：
  - 缺光：茎细长弯曲（徒长）→叶片发黄→不开花
  - 缺水：叶片下垂→边缘枯焦→整株萎蔫
  - 低温：叶片变紫→生长停滞→冻死

#### 🍓 草莓 — 难度 ★★★★
- **适宜温度**：15~25°C
- **耐热**：怕热，>28°C 停止开花，>32°C 持续数天叶片灼伤
- **耐寒**：轻微耐寒，-5°C 以下地上部冻死（根可能存活）
- **需水**：高，但极怕涝；土壤湿度 50~70% 最佳
- **日照**：需要充足日照才能结果
- **生长周期**：90~120 天结果
- **寿命**：2~3 年
- **真实反应**：
  - 高温：先停止开花→叶缘焦枯→果实变小→持续高温根部损伤→死亡
  - 涝水：根系窒息→叶片萎蔫（看似缺水实为烂根）→灰霉病→死亡
  - 干旱：叶片卷曲下垂→果实干瘪→整株枯萎
  - 缺光：徒长不开花，结果少

#### 🌸 樱花 — 难度 ★★★★★
- **适宜温度**：15~25°C（生长期）
- **需要冬季低温**：必须经历一段 <7°C 的低温期（春化），否则不开花
- **耐热**：>35°C 叶片灼伤
- **耐寒**：落叶后耐寒到 -15°C
- **需水**：中等；怕涝
- **日照**：需要良好日照
- **生长周期**：树苗→开花需 3~5 年（游戏中加速为 365 天）
- **寿命**：数十年
- **季节性**：春天开花→夏天绿叶→秋天变色落叶→冬天休眠
- **真实反应**：
  - 没经历冬天：春天不开花（致命体验打击）
  - 夏季高温：叶片灼伤→早落叶
  - 积涝：根部腐烂→整树衰弱

### 地块系统
- 起步 1 块地，养成一棵植物后可选择领取新地块
- 每块地独立管理（独立植物、独立浇水）
- 所有地块的植物都需要照顾
- 不想要可以不领新地块

### 操作
- **浇水** — 给选中的地块浇水，增加土壤湿度
- **收获** — 成熟的果实/花朵可以收获（纯记录，无金币）
- **换种** — 挖掉当前植物，地块空出来重新选种
- **查看** — 查看植物状态、环境数据、健康详情

## 技术方案

### 天气 API
- **Open-Meteo**（免费，无需 API Key）
  - 当前温度、降水、天气状况
  - 每小时/每天更新
- 用户授权地理位置，或手动选城市
- 离线时使用缓存的最近天气数据

### 引擎
- **Cocos Creator 3.x**（如已安装）
- 或 **微信小游戏原生 Canvas + TypeScript**（更轻量，待定）

### 核心数据模型
```typescript
interface PlantInstance {
  id: string
  type: PlantType              // 幸运草/向日葵/草莓/樱花
  plantedAt: number            // 播种时间戳
  healthState: HealthState     // 健康/轻微受损/明显受损/严重衰弱/死亡
  healthValue: number          // 0~100 精细健康值
  growthProgress: number       // 0~1 生长进度
  lastWateredAt: number        // 上次浇水时间
  harvestCount: number         // 收获次数
  stressDays: Record<string, number>  // 各类胁迫累计天数
}

interface EnvironmentState {
  temperature: number          // 当前气温 °C
  rainfall: number             // 降水量 mm
  sunlight: number             // 日照强度 0~1
  soilMoisture: number         // 土壤湿度 0~100%
  weatherCode: number          // 天气状况代码
  updatedAt: number            // 数据更新时间
}
```

### 模拟引擎
```typescript
// 每次打开游戏，补算离线期间的状态变化
function simulateOfflineGrowth(plant: PlantInstance, weatherHistory: WeatherData[]) {
  for (const day of weatherHistory) {
    // 1. 更新环境参数
    updateSoilMoisture(plant, day)
    // 2. 检查各类胁迫
    checkTemperatureStress(plant, day)
    checkWaterStress(plant, day)
    checkLightStress(plant, day)
    // 3. 更新健康状态
    updateHealthState(plant)
    // 4. 更新生长进度（受健康影响）
    updateGrowth(plant, day)
    // 5. 检查死亡
    if (plant.healthValue <= 0) {
      plant.healthState = 'dead'
      break
    }
  }
}
```
